# StochOpt_Code

This is the code used in the generation and postprocessing of the data published in the dissertation "Stochastic Optimization with High-Dimensional Uncertainties" <doi here. \n
It includes the code fo reach numerical simulation presented by section and subsegmented into each example. Each folder contains an additional readme file to explain how it works. \n
Additionally, data generated in the dissertation is available on Zenodo at 10.5281/zenodo.18714508 . The data files are also structured in the same folder structure as the code.
